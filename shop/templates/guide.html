Application Guide

Creating a User

  1. Making an order requires a customer ID.
  2. To create a new customer, you can use the webapp to login via OpenID first.
  3. On sign-in, the user can choose to be admin or normal.
  4. If a user switches to admin, they will be prompted to enter the password: abcdef.
  5. On first sign-in, the user will also be prompted to add a phone number.
  6. All admin users with valid emails are notified via email when an order is made.
  7. One can view all users via the REST API.

API Endpoints

Full REST API Documentation
