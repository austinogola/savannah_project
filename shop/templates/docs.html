API Documentation

BASE_URL - https://savannah.austino.online

GET {BASE_URL}/api/categories/

Fetch all categories.

GET {BASE_URL}/api/categories/{id}/

Fetch details of a category by ID.

POST {BASE_URL}/api/categories/

Create a new category.

Body Parameters:
{
  "name": "string" (required),
  "description": "string" (optional)
  "parent": "string" (optional) - name of the parent category it belongs to
}
GET {BASE_URL}/api/products/

Fetch all products.

POST {BASE_URL}/api/products/

Create a new product. (One of category_id or category_name required)

Body Parameters:
{
  "name": "string" (required),
  "price": "decimal" (required),
  "category_id": "int" (optional),
  "category_name": "string" (optional),
}
POST {BASE_URL}/api/orders/

Create a new order.

Body Parameters:
{
  "customer_id": "int" (required),
  "items": [
    {"product_id": int, "quantity": int},
    ...
  ],
}
GET {BASE_URL}/api/customers/

Fetch all customers.
