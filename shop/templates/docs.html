Savannah API Documentation

Categories

  • GET /api/categories/ → List all categories
  • POST /api/categories/ → Create category
  • GET /api/categories/{id}/ → Retrieve category
  • GET /api/categories/{id}/avg-price/ → Get average product price

Products

  • GET /api/products/ → List all products (filters: category_id, category_name)
  • POST /api/products/ → Create a new product

Customers

  • GET /api/customers/ → List customers (filters: phone, user_id)
  • POST /api/customers/ → Create a customer

Users

  • POST /api/users/ → Create user (also auto-creates Customer)
    • {
  "username": "jdoe",
  "password": "mypassword",
  "first_name": "John",
  "last_name": "Doe",
  "phone": "+123456789"
}

Orders

  • POST /api/orders/ → Create order
